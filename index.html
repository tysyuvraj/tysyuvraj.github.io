Selected Work

Master's thesis · ongoing

Anticipatory Multimodal Intent Prediction for Real-Time HRI

A real-time supervisor for autonomous robot manipulation. It watches video and audio, predicts human intent 1–2 seconds ahead, and decides when a running NVIDIA GR00T policy should stop, switch tasks, or continue. A single unmodified "brain" was validated across two structurally different robots and three independently trained policies — coupled to each through a fixed 10-method contract.

How it's built

Engineering Highlights

H1

Unified multimodal perception

One model (Qwen3-Omni, via vLLM) handles video and audio jointly, replacing what would otherwise be separate VAD, ASR, and vision-encoder components.

H2

Diagnosing a production bug

Traced a ~50% output parse-failure rate to a known vLLM/Qwen3 compatibility issue and validated a fix across the full deployment — from guesswork to a reproducible root cause.

H3

Model-behavior-driven design

The model is unreliable at structured fields but reliable at focused yes/no questions. All control logic lives in software that never trusts the model's raw decision directly.

Stated honestly

Known Limitations

Open-vocabulary understanding, closed-vocabulary execution. The robot only performs its trained skills; adding one requires new demonstration data.

The model is reliable at focused yes/no visual checks but weak at fine-grained grounding such as color naming — the perception design works around this rather than solving it.

The anticipatory speech path trades selectivity for speed: it can fragment compound commands and needs confirmation gates that add back some latency.

About
